1 Model Evaluation Tools Plus (METplus) Repository
2 ================================================
3 
4 Welcome to the METplus GitHub repository hosted to the community through
5 the Developmental Testbed Center (DTC).
6 
7 METplus is a Python scripting infrastructure around the MET verification tools
8 (and eventually METViewer, a tool used for plotting MET output verification statistics).
9 
10 Documentation for both Users and Contributors can be found here: https://ncar.github.io/METplus.
11 Fore more information about the entire suite of tools, please visit the MET website
12 (https://dtcenter.org/community-code/model-evaluation-tools-met) and direct questions
13 to met_help@ucar.edu.
14 
15 This infrastructure utilizes the NCEP produtil package, which is a platform-independent
16 weather and ocean forecasting utility package developed at the National Oceanic
17 and Atmospheric Administration (NOAA).
